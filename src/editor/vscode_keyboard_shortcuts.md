### 代码浏览

vscode 菜单栏的 **go** 选项

| cmd                   | keyboard       | content               | 图示  |
| --------------------- | -------------- | --------------------- | --- |
| trigger suggest       | Ctrl + i       | 写代码时触发代码补全提示框         |     |
| go to Definition      | F12            | 查看变量定义                |     |
| back                  | Ctrl + Alt + - | 返回原来位置                |     |
| go to Reference       | Shife + F12    | 查看变量在哪些地方被引用(包括调用和实现) |     |
| go to implementations | Ctrl + F12     | 查看变量实现的定义             |     |
